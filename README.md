# Pillsgood.Unity.ScriptGenerator
## Installation
add to ```Packages/manijest.json```
```json
{
 "com.pillsgood.scriptgenerator": "https://github.com/pillsgood/Pillsgood.Unity.ScriptGenerator.git"
}
```

or install from git url (https://github.com/pillsgood/Pillsgood.Unity.ScriptGenerator.git) using package manager.

## Supported
* AnimatorController
  * Parameters (float/bool/int)
  * Parameters: triggers
  * Layers
* Sprite Library Asset
  * get sprite from index in statically typed categories
  * get sprite from enum for category labels
  
## Requirement
* Unity 2021.2 (Todo: support older versions)
